## üéØ Learning Paths

### Path 1: Complete Beginner (45 minutes)

**Goal**: Send your first batch of personalized emails

1. Read [QUICK-START.md](QUICK-START.md) - 10 min
2. Download tool and example file - 2 min
3. Follow Quick Start steps - 10 min
4. Practice with your own data - 15 min
5. Read README "Common Use Cases" section - 8 min

**Result**: Confident with basic email generation

### Path 2: Power User (2 hours)

**Goal**: Master all features including merge modes and templates

1. Complete Beginner path - 45 min
2. Read full [README.md](README.md) - 30 min
3. Try each merge mode - 15 min
4. Create and save templates - 10 min
5. Practice with filters - 10 min
6. Read [TEMPLATES.md](TEMPLATES.md) - 10 min

**Result**: Expert user who can handle complex scenarios

### Path 3: Template Master (1 hour)

**Goal**: Create reusable templates for common emails

1. Read [QUICK-START.md](QUICK-START.md) - 10 min
2. Read [TEMPLATES.md](TEMPLATES.md) - 15 min
3. Study template examples - 15 min
4. Create 3 custom templates - 20 min

**Result**: Template library for recurring email needs

### Path 4: Accessibility Expert (1 hour)

**Goal**: Understand and use all accessibility features

1. Read [ACCESSIBILITY.md](ACCESSIBILITY.md) - 20 min
2. Test with keyboard only - 10 min
3. Test with screen reader - 20 min
4. Review accessibility checklist - 10 min

**Result**: Confident using tool with assistive technology

**Goal**: Ready to contribute code or documentation

1. Read [CONTRIBUTING.md](CONTRIBUTING.md) - 15 min

3. Make test change - 10 min

## ‚ùì FAQ Quick Links

### Common Questions

**Q: Can I use this offline?**
A: Yes! Once downloaded, it works completely offline. See [README](README.md#-privacy--security)

**Q: What file formats are supported?**
A: CSV, XLSX, and XLS. See [File Upload Guide](README.md#1-preparing-your-data-file)

**Q: How many emails can I generate?**
A: Limited only by browser memory. Tested up to 10,000 rows. See [Technical Details](README.md#-technical-details)

**Q: Is my data secure?**
A: Yes, everything runs locally. No data is uploaded. See [Privacy & Security](README.md#-privacy--security)

**Q: Can I save my templates?**
A: Yes! Use the template management feature. See [Template Management](README.md#10-template-management)

**Q: How do I use keyboard shortcuts?**
A: Press `?` to see help panel. See [Keyboard Shortcuts](README.md#11-keyboard-shortcuts)

**Q: What browsers are supported?**
A: Chrome, Firefox, Safari, Edge (90+). See [Browser Compatibility](README.md#browser-compatibility)

**Q: Can I undo if I make a mistake?**
A: Yes! Press Ctrl+Z or click undo button. See [Undo Feature](README.md#12-undo-feature)

**Q: How do I report a bug?**
A: Open a GitHub issue. See [Reporting Bugs](CONTRIBUTING.md#1-report-bugs)

**Q: Can I contribute?**
A: Yes! We welcome contributions. See [CONTRIBUTING.md](CONTRIBUTING.md)

## üìã Cheat Sheets

### Keyboard Shortcuts
```
Ctrl/Cmd + G    Generate emails
Ctrl/Cmd + P    Preview email
Ctrl/Cmd + S    Save template
Ctrl/Cmd + Z    Undo
?               Help panel
Esc             Close modal
```

### Variable Syntax
```
{{ColumnName}}     Replace with data from ColumnName
{{First Name}}     Works with spaces
{{Email}}          Common: Email address
{{Name}}           Common: Person's name
{{Company}}        Common: Company name
```

### Merge Mode Selection
```
No merging         ‚Üí One email per row
Merge recipient    ‚Üí Combine rows with same email
Multi-recipient    ‚Üí All recipients in one email
```

### Filter Operators
```
Equals            ‚Üí Exact match
Contains          ‚Üí Partial match
Not Empty         ‚Üí Has any value
Empty             ‚Üí Has no value
```
